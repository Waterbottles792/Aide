# Aide

> **An AI mentor for cybersecurity learning that teaches instead of spoiling.**

Aide is a GenAI-powered assistant designed for cybersecurity learners using platforms like **TryHackMe**, **Hack The Box**, **PortSwigger Web Security Academy**, and CTFs.

Unlike traditional AI assistants that immediately provide solutions, Aide provides **progressive hints**, contextual explanations, and guided assistance to help users learn while solving challenges.

---

## Features

* Progressive, anti-spoiler hint system
* Multi-provider LLM support
* Bring your own API keys
* Context-aware conversations
* Multiple learning modes
* Chat history and session management
* DuckDuckGo-powered educational resource retrieval
* Modern, responsive interface

---

## Supported AI Providers

* OpenAI
* Anthropic Claude
* Google Gemini
* Groq
* OpenRouter
* Ollama *(planned)*

---

## Philosophy

Aide is built around a simple principle:

> **Teach. Don't Spoil.**

Instead of revealing answers immediately, Aide encourages critical thinking by adapting its guidance to your progress and providing increasingly detailed hints only when needed.

## Getting Started

Clone the repository:

```bash
git clone https://github.com/Waterbottles792/Aide.git
cd Aide
```

Install the required dependencies and configure your preferred AI provider using your own API key.
