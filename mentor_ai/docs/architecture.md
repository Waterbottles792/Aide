# Architecture (Phase 1)

This document sketches the high-level architecture for the Mentor AI project. Phase 1 focuses on a local FastAPI backend and a placeholder frontend that will later become a floating overlay.

Components:
- Backend: FastAPI (chat endpoint, provider abstraction)
- Frontend: React/Electron or Tauri (placeholder)
- Core modules: llm router, hints, rag — stubbed in Phase 1
