# Mentor AI Development Plan (Phase 1)

## Vision

Build an AI desktop mentor for cybersecurity learning platforms that teaches through progressive hints rather than giving direct solutions.

## Phase 1 — Core App (this sprint)

- Desktop overlay skeleton (Electron/Tauri) — placeholder frontend
- FastAPI backend with a chat endpoint and provider abstraction
- Settings page and API key storage (UI placeholder + backend endpoint)
- Theme support scaffolding
- Provider-agnostic LLM router (stubbed)
- Basic docs and README

## Completion Criteria

- Backend responds to `/chat` with a structured response.
- Frontend skeleton can connect to backend (placeholder README and instructions).
